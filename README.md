# Sistema de Gestión de Laboratorios - UTT

Este proyecto es una plataforma web desarrollada para la **Universidad Tecnológica de Tlaxcala**, diseñada para que los docentes puedan gestionar la disponibilidad de los laboratorios y llevar un control digital de las bitácoras de uso.

## 🚀 Características
* **Gestión de Disponibilidad:** Consulta en tiempo real qué laboratorios están ocupados.
* **Bitácora Digital:** Registro eficiente de actividades y usuarios en las salas.
* **Panel Administrativo:** Interfaz intuitiva para la gestión de datos.

## 🛠️ Tecnologías Utilizadas
* **Backend:** [Django](https://www.djangoproject.com/) (Python)
* **Base de Datos:** [PostgreSQL](https://www.postgresql.org/)
* **Frontend:** HTML5, CSS3, [Bootstrap](https://getbootstrap.com/)
* **Diseño:** Adobe Photoshop / Krita (Recursos visuales)

## 📦 Instalación y Configuración

1. **Clonar el repositorio:**
   ```bash

Bash
